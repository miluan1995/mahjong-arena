# 🀄 Mahjong Arena — 项目设计文档

> OpenClaw AI 麻将竞技平台 | 不发币，纯游戏

## 核心定位

**让 AI Agent 打麻将，让人类看得爽、玩得爽。**

## 两种模式

### 1. Agent vs 人类场
- 1-3 个 AI Agent + 人类玩家凑满 4 人
- 人类通过网页端操作（点牌出牌、吃碰杠胡）
- AI 实时决策，有思考过程展示（"黑瞎子在思考..."）
- 难度可调（新手/普通/大师）

### 2. Agent 专场（观战模式）
- 4 个 AI Agent 自动对打
- 人类纯观看，像看棋赛直播
- 实时解说弹幕（AI 自动解说关键操作）
- 每个 Agent 有个性（激进型/保守型/计算型/运气型）
- **这个模式成本最低，开发最快，观赏性最高**

## 成本控制策略

### 选 MVP 最小方案
- **规则：川麻血战到底** — 规则简单好理解、节奏快、观赏性强
- **前端：纯 HTML + Canvas** — 不用 React/PixiJS，单文件部署，零依赖
- **后端：无** — 纯前端跑规则引擎，AI 策略本地计算
- **AI 接入：规则型 AI 先行** — 不调 LLM，用算法策略，零 API 成本
- **部署：GitHub Pages** — 免费

### 后期可加（不在 MVP）
- WebSocket 多人在线
- OpenClaw Agent 真正接入（通过 API 调用 LLM 决策）
- 排行榜/段位系统
- 回放系统

## 川麻血战规则（简版）

- **牌种**: 万、条、筒 各 4×9 = 108 张（无字牌无花牌）
- **玩法**: 血战到底 — 一人胡了不结束，继续打到只剩一人
- **胡牌**: 基本胡（对对胡、清一色、七对等）
- **特殊**: 刮风下雨（杠上开花）、自摸加倍
- **核心交互**: 摸牌 → 出牌 → 其他玩家判断吃/碰/杠/胡 → 下家

## 渲染设计

### 牌面：程序化绘制（零素材成本）
- Canvas 绘制白色圆角矩形 + 文字/简笔画
- 万：红色数字 + "万"
- 条：绿色竹节图案
- 筒：蓝色圆形图案
- 效果：适当阴影 + 倒映，够精致

### 牌桌布局
```
        ╔══════════════════╗
        ║   对面玩家(北)    ║  ← 牌背朝下
        ╠══════════════════╣
   左   ║                  ║  右
   家   ║     牌  桌       ║  家
  (西)  ║   (弃牌区域)     ║ (东)
        ║                  ║
        ╠══════════════════╣
        ║   自己/观战(南)   ║  ← 牌面朝上
        ╚══════════════════╝
```

### 观战模式特色
- 所有玩家牌面可见（上帝视角）
- 实时出牌高亮 + 路径动画
- 自动解说："黑瞎子打出 3万，这是在做清一色！"
- AI 思考气泡："🤔 要不要碰..."
- 关键操作慢动作回放

## AI 个性系统

| Agent 名 | 风格 | 策略 |
|----------|------|------|
| 黑瞎子 🐻 | 激进 | 大牌优先，敢赌敢冲 |
| 铁柱 🔨 | 保守 | 小胡为主，稳扎稳打 |
| 算盘 🧮 | 计算 | 最优解，效率至上 |
| 锦鲤 🐟 | 混沌 | 随机出牌但运气逆天 |
